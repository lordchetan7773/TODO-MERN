Features
--------------

- User authentication with persistent Todos
- Responsive TailwindCSS design
- RESTful API
- Body / Params type checking
Getting Started
----------

1. Clone / Download the repository

1. Install dependencies

   ```bash
   yarn install
   ```

1. Build the project

   ```bash
   yarn build
   ```

1. Run the project

   ```bash
   yarn start
   ```

1. (Run only client)

   Navigate to `/client`
   ```bash
   yarn install
   yarn dev
   ```
